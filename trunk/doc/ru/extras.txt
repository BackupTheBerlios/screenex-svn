    Дополнительные возможности

    screenex-helper
Данный файл является оберткой для связки GNU Screen и Screenex, он автоматически 
запускает GNU Screen и настраивает его для работы со Screenex. screenex-helper может 
непосредственно использоваться в качестве user shell. В данном сценарии 
использования возможно построение своеобразного многопользовательского терминального 
сервера доступа.
